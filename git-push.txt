QUICK REFERENCE COMMANDS

- Check status
git status

- Add all changes
git add .

- Commit with message
git commit -m "Your message here"

- Push to GitHub
git push

- View commit history
git log --oneline

- Undo last commit (keep changes)
git reset --soft HEAD~1

- Discard local changes
git checkout -- filename.py

- Pull latest from GitHub
git pull
